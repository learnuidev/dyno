# Dyno

dyno is an open source next gen dynamodb client

## Prerequisites

### 1. Populate `.env.local`

```sh
# AWS
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=

```

This is need to talk to dynamodb

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Features

0. Search
1. Clone
2. Clear
3. ETL
4. Dynolog
5. Integrations (google spreadsheet, airtable)
6. History
